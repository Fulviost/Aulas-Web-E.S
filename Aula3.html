Nesta aula apredemos sobre listas. Há 3 tipos de listas: "Lista de Definição", "Lista não Enumerada" e "Listas Numeradas".

Sobre a Lista de Definição:

Usamos a TAG ‹dl› para falar o tipo de lista.

A TAG ‹dt› é o termo a ser definido.

É usado ‹dd›para definir o termo.

Exemplo: 
			‹dl›.
				‹dt› WebDesign ‹⁄dt›.
				‹dd›Criação de páginas‹⁄dd›.

Sobre a lista não numerada:

Usamos a TAG ‹ul› para falar o tipo de lista.

A TAG ‹li› é utilizado para cada termo que você for definir.

Exemplo: 
		‹ul›
 		  ‹li›Dom Casmurro‹⁄li›.
 		  ‹li›O Cortiço‹⁄li›.
		  ‹li›A Moreninha‹⁄li›.

Observação: Podemos mudar o formato da bolinha que será imprimida no site utilizando ‹ul type="circle", "disc" e "square".›

Sobre as Listas Numeradas:

Usamos a TAG ‹ol› para falar o tipo de lista.

A TAG ‹li› é utilizado para cada termo que você for definir.

Observação: Ao invés de mudarmos o formato da bolinha agora podemos colocar letras utilizando ‹ol type="a"› e para definir qual dos termos vai começar a contar é só utilizar o atributo start.

Exemplo:
			‹ol type="a" start="3"›

Para utilizar nas listas:

Para abrir uma nova aba clicando em uma referência utilizamos o atributo target="_blank".

Exemplo:
 		‹a href="nomedaimagem.jpg" target="_blank"›
Para mudar a cor da referência antes e depois de clicar basta utilizar o comando:
		‹body link="yellow" vlink="blue"›

A TAG ‹p› é utilizada para criação de parágrafos. Ela também cria uma maior distância de uma linha para outra.

 			‹p› Insira o texto desejado aqui.

A TAG ‹pre› é utilizada para para identar a página da mesma maneira que está escrita no código html.
