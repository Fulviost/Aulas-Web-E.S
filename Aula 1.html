Introdução à HTML5

Este é um resumo da aula do dia 14/08/2018

Toda vez que forem inciar a aula na Janaína é necessário colocar os seguintes códigos:

‹!doctype html›
‹head›

‹/head›
‹body›

‹/body›
‹/html›


Dentro de uma tag você pode ter a TAG a propriedade e o atributo.

Exemplo: ‹html› é a TAG.
lang é a propriedade.
"pt-br" é o atributo.
‹meta lang="pt-br"›


Para salvar o arquivo como página inical é necessário colocar a palavra "index.html" no arquivo.



Motivo: O servidor só reconhece quando há o index pois ela será a página principal do site.

No HTML não há depurador, ou seja, não tem como saber qual linha a menos que você procure.


A TAG "‹p›" serve para escrita dentro da página.

Exemplo: ‹p› "Texto desejado".



A TAG "‹h1›" serve para criar um cabeçalho do maior tamanho possível.

Explicação: Há 6 tipos de tamanho do cabeçalho, o h1 é o maior deles, o h6 é o menor, escolha o desejado para sua página.



Há outros atributos que são bastante úteis como por exemplo:

A propriedade "align" serve para colocar o cabeçalho no lugar desejado na página.


Os atributos "Right","Center" e "Left" servem para colocar o cabeçalho em um lugar específico na página.



A TAG ‹font› serve para escolher a fonte, o tamanho e a cor de determinado texto.


Exemplo: ‹font face="Arial" size="7" color="blue"›


"face" é para fonte.


"size" é para o tamanho das letras.


"color" é para determinar a cores das palavras.
